# Repository screenshots

Place three PNG screenshots in this directory using these exact filenames:

1. `01-ui-overview.png` — the main NUST loan eligibility interface.
2. `02-model-result.png` — a completed assessment showing the gauge and factor chart.
3. `03-docker-running.png` — Docker Desktop or terminal output showing the web and API containers running.

Recommended image width: 1400–1600 pixels. Crop private information before committing screenshots.
